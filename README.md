**WOW** is a modern and responsive flight booking website built using HTML, CSS, JavaScript, and Bootstrap. It showcases clean UI, smooth navigation, and interactive alerts using SweetAlert.

##  Technologies Used

- HTML5  
- CSS3

- Bootstrap 4

-  About  
-  Gallery
